# Project Setup

## REQUIREMENTS

- Node.js 18+ (or later)
- npm 10+ (or yarn)

## HOW TO INSTALL

1. Clone project
```bash
git clone https://github.com/your-user/aptech1-midterm-exam.git
cd aptech1-midterm-exam
```
2. Install deps
```bash
npm install
```
3. Run dev server
```bash
npm run dev
```
4. Open browser:
- http://localhost:5173

## USAGE

- `/profile` shows blank profile for not signed-in user
- `/profile/:username` shows dummy profile data from `src/pages/Profile.jsx`

## QUICK COMMANDS

- `npm run lint`
- `npm run build`
